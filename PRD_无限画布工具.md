## 三、文档说明

### 3.1 文档目的
本文档用于描述「容量AI漫剧平台」创作引擎画布（canvas.html）的功能需求、交互逻辑和验收标准，供产品、设计、研发、测试团队参考。

### 3.2 适用范围
- 前端开发工程师
- UI/UX设计师
- 测试工程师
- 产品经理

### 3.3 名词解释

| 名词 | 解释 |
|------|------|
| 无限画布 | 基于ReactFlow的无限延伸画布，支持节点拖拽、连线、缩放 |
| 节点 | 画布上的功能单元，代表AI创作流程中的一个步骤 |
| 文本节点 | 输入剧本/文本内容，触发后续AI生成流程的起点节点 |
| 角色生成 | 基于文本节点输入，AI生成角色形象资产 |
| 场景生成 | 基于文本节点输入，AI生成场景概念图 |
| 道具生成 | 基于文本节点和场景输入，AI生成道具资产 |
| 分镜生成 | 综合剧本、角色、场景、道具，AI生成分镜脚本 |
| 视频合成 | 将分镜和视频片段合成为最终成片 |
| 历史资产 | 用户过往生成的AI资产（角色/场景/道具/分镜/视频）的归档管理 |
| 顶部工具栏 | 节点选中时悬浮显示的操作工具栏（图片编辑、下载等） |
| 底部工具栏 | 画布底部状态栏，含节点统计、画布管理、快捷键、历史记录入口 |

## 四、需求背景

### 4.1 产品现状
容量AI漫剧平台面向短剧创作者，提供从剧本到成片的AI辅助创作全流程。当前核心痛点：
- 创作者需要手动管理多个AI工具（文生图、图生视频、视频合成）
- 缺乏可视化的创作流程管理，资产散落在各处
- 无法直观查看各环节的依赖关系和执行状态

### 4.2 竞品分析

| 竞品 | 画布功能 | 节点类型 | 资产管理 | 我们的优势 |
|------|----------|----------|----------|------------|
| ComfyUI | 有，技术向 | 通用AI节点 | 无 | 更贴近短剧创作场景 |
| InvokeAI | 有，偏设计 | 图像生成节点 | 基础 | 支持视频合成链路 |
| 即梦 | 无 | 单点工具 | 有 | 全流程可视化 |
| 可灵 | 无 | 单点工具 | 有 | 节点间自动连线 |

### 4.3 用户故事

> 作为短剧创作者，我希望在可视化画布上拖拽搭建AI创作流程，以便直观管理从剧本到成片的每个环节，减少工具切换成本。

> 作为美术设计师，我希望在画布上直接编辑生成的角色/场景图片，以便快速迭代资产质量。

> 作为项目经理，我希望查看历史生成的所有AI资产，以便复用和归档管理。

## 五、需求范围

### 5.1 In-Scope（范围内）
- 8种节点类型的画布可视化（文本节点、角色生成、场景生成、道具生成、分镜生成、视频生成、图片生成、视频合成）
- 节点间的自动连线与类型校验
- 节点选中时的顶部工具栏操作（图片编辑、下载、本地上传、资产库导入）
- 底部工具栏（节点统计、画布管理、快捷键、历史记录）
- 历史资产全屏弹窗（分类筛选、日期分组、多选操作）
- 暗色主题UI

### 5.2 Out-of-Scope（范围外）
- 后端AI模型调用（仅前端原型）
- 用户登录/权限管理
- 实时协作编辑
- 移动端适配
- 成片节点的视频预览和编辑功能（已删除）

#### 6.1.2 画布基础操作

| 操作 | 触发方式 | 说明 |
|------|----------|------|
| 添加节点 | 底部工具栏「+」按钮 / 右键菜单 / 拖拽面板 | 在画布中心位置创建新节点 |
| 删除节点 | 右键菜单「删除节点」/ Delete键 | 删除选中节点及其关联连线 |
| 复制节点 | 右键菜单「复制节点」/ ⌘+拖动 | 创建选中节点的副本 |
| 连接节点 | 拖拽输出端口到目标节点输入端口 | 自动校验类型兼容性 |
| 缩放画布 | ⌘+滚轮 / 触控板双指捏合 | 支持0.1x~4x缩放 |
| 拖动画布 | Space+拖拽 / 触控板双指滑动 | 平移视图 |
| 适应画布 | ⌘+0 | 自动缩放至全部节点可见 |
| 撤销/重做 | ⌘+Z / ⌘+⇧+Z | 最多保存40步历史 |

### 6.3 功能说明

#### 6.3.1 节点卡片交互

**FE-01 文本节点**
- 卡片显示：图标、标签、状态指示器
- 点击卡片：展开/收起详情面板
- 详情面板：模型选择、风格选择、描述输入、成本估算
- 状态流转：ready → queued → running → completed/failed

**FE-02 角色生成节点**
- 卡片显示：角色预览图（如有）、角色名称、状态
- 选中时顶部工具栏：图片编辑（裁剪、重绘、抠图、超分、擦除、去水印、扩图）、本地上传、资产库导入、下载
- 图片编辑下拉菜单：默认收起，点击展开
- 点击卡片展开面板：角色名称、外貌描述、出场集数、模型选择、比例、分辨率、风格

**FE-03 场景生成节点**
- 同角色生成节点结构，标签为「场景生成」
- 面板字段：场景名称、场景描述、模型选择、比例、分辨率、风格

**FE-04 道具生成节点**
- 同角色生成节点结构，标签为「道具生成」
- 面板字段：道具名称、道具描述、模型选择、比例、分辨率、风格

**FE-05 分镜生成节点**
- 卡片显示：分镜缩略图网格（最多4张）、状态
- 选中时顶部工具栏：图片编辑、本地上传、资产库导入、下载
- 面板字段：分镜名称、分镜描述、模型选择、比例、分辨率、风格

**FE-06 视频生成节点**
- 卡片显示：视频时长、分辨率、状态
- 面板字段：模型选择、时长、分辨率、风格

**FE-07 图片生成节点**
- 同角色生成节点结构，标签为「图片生成」
- 面板字段：图片名称、描述、模型选择、比例、分辨率、风格

**FE-08 视频合成节点**
- 卡片显示：合成片段列表、总时长、状态
- 点击卡片：打开视频合成编辑弹窗
- 弹窗功能：片段列表（拖拽排序）、添加片段、删除片段、导出到本地
- 导出到本地：提示「导出到本地功能开发中」

#### 6.3.2 底部工具栏

| 元素 | 说明 |
|------|------|
| 节点统计 | 显示当前画布节点总数、连线数、选中数 |
| 添加节点按钮 | 圆形「+」按钮，点击展开添加节点面板 |
| 画布管理 | 点击切换画布菜单显示/隐藏 |
| 快捷键 | 点击打开快捷键说明弹窗 |
| 历史记录 | 点击打开历史资产全屏弹窗 |
| 自动保存状态 | 显示「就绪」绿色指示器 |

#### 6.3.3 历史资产弹窗

| 区域 | 说明 |
|------|------|
| 顶部标题栏 | 「历史资产」+ 缩放控制（-/100%/+）+ 关闭按钮 |
| 分类筛选 | 全部、角色生成、场景生成、道具生成、分镜生成、视频合成 |
| 内容区 | 按日期分组缩略图网格，支持复选框多选 |
| 操作栏 | 已选N项、删除、下载、使用、取消选择 |
| 主题 | 暗色主题，与画布风格一致 |
| 渲染方式 | ReactDOM.createPortal挂载到document.body，zIndex: 99999 |

#### 6.3.4 添加节点面板

| 元素 | 说明 |
|------|------|
| 触发方式 | 底部工具栏「+」按钮 / 右键菜单 |
| 面板位置 | 底部居中弹出，fixed定位 |
| 节点列表 | 拖拽或点击添加，显示图标、标签、副标题 |
| 副标题 | 文本节点：文本解析、角色设定；角色生成：角色形象、三视图；场景生成：场景概念、环境设定；道具生成：道具资产、物品生成；分镜生成：分镜脚本、镜头设计；视频合成：视频合成 |
| 上传模块 | 底部「上传」入口，支持图片、视频等素材上传 |

## 七、非功能需求

### 7.1 性能要求

| 指标 | 要求 |
|------|------|
| 首屏加载 | ≤ 3秒（含CDN资源加载） |
| 节点渲染 | 50个节点流畅操作，无明显卡顿 |
| 画布缩放 | 60fps，无掉帧 |
| 历史资产弹窗 | 首次打开 ≤ 1秒，滚动流畅 |

### 7.2 兼容性

| 浏览器 | 版本要求 |
|--------|----------|
| Chrome | ≥ 100 |
| Safari | ≥ 15 |
| Edge | ≥ 100 |
| Firefox | ≥ 100 |

### 7.3 安全要求
- 所有外部资源通过 HTTPS 加载
- 不支持用户上传文件的实际存储（前端原型）

### 9.2 风险项

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| ReactFlow性能瓶颈（大量节点） | 高 | 虚拟渲染、节点懒加载 |
| 图片编辑工具集成复杂度 | 中 | 优先使用第三方库（如Cropper.js） |
| 历史资产数据量大 | 中 | 分页加载、虚拟滚动 |
| 暗色主题视觉一致性 | 低 | 建立设计Token规范 |
